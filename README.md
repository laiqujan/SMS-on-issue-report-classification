# SMS on Issue Report Classification
This repository contains a replication package for our study: "Automatic techniques for issue report classification: A systematic mapping study."
## Repository Content
The dataset consists of three Excel files:
### 1. `Final_map.xlsx`
This file provides the complete mapping of the selected studies, including all extracted data.
It also contains:
- A simplified version of the final map.
- Detailed classifications of the extracted data (e.g., classification of automatic techniques).
### 2. `Master_sheet.xlsx`
This file summarizes the study selection process, including:
- The total number of papers initially identified
- The number of papers remaining after duplicate or conference proceeding removal
- The number of papers retained after applying the inclusion and exclusion criteria
### 3. `List_of_excluded_studies.xlsx`
This file contains the list of studies that were excluded during the selection process.
